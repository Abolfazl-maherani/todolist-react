## Live Demo

### Prerequisites

- Node.js
- npm or Yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Abolfazl-maherani/todolist-react.git
   
2. Install dependencies:
  ```npm install```
  ```yarn install```
3. Start the development server:
  ```npm start```
  ```yarn start```
